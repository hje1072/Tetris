## 자바로 구현한 테트리스 게임
![Main](https://github.com/hje1072/Tetris/assets/120327265/ca286a79-8741-405b-a988-ecbeccdc52c4)

### 기능
- 일반 모드
- 아이템 모드 (현재 구현중에 있습니다)
- 색맹자를 위한 색 설정(현재 구현중에 있습니다)

### 점수
- 각 줄 제거 시 10점 획득
- 점수판에는 상위 랭크 10등까지만 기록

### 규칙
- 일반적인 테트리스 규칙을 따릅니다
- 블럭은 랜덤하게 생성됩니다
